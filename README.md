# Bias Propagation Analysis in AI Chatbots

This project is part of a research study on evaluating bias and fairness in AI chatbot responses using prompt-based analysis and sentiment evaluation.

## Project Overview

The goal of this project is to analyze chatbot responses and determine whether they exhibit bias. As part of the evaluation, sentiment analysis is performed on responses generated by different AI chatbots.

The chatbots analyzed include:
- ChatGPT
- Gemini
- Perplexity
- Copilot

## Methodology

1. A dataset of prompts was created.
2. Responses from multiple chatbots were collected.
3. Sentiment analysis was performed using TextBlob.
4. Each response was classified as:
   - Positive
   - Neutral
   - Negative

This sentiment data supports bias and fairness evaluation.

## 🛠️ Technologies Used

- Python
- Pandas
- TextBlob
- Excel (for dataset storage and outpu)


 
## ⚙️ How to Run the Project

### Step 1: Install Required Libraries

Run the following command in terminal:
pip install pandas textblob openpyxl


Also download TextBlob corpora:
python -m textblob.download_corpora


---

### Step 2: Prepare Input File

- Place your dataset file `BiasAnalysis.xlsx` in the same folder.
- Ensure it contains the following columns:
  - ANSWERS - CHATGPT
  - ANSWERS - GEMINI
  - ANSWERS - PERPLEXITY
  - ANSWERS - COPILOT

---

### Step 3: Run the Script
python sentiment_analysis.py



---

### Step 4: Check Output

- A new file `Final_Output.xlsx` will be generated.
- It will include sentiment columns:
  - Sentiment_ChatGPT
  - Sentiment_Gemini
  - Sentiment_Perplexity
  - Sentiment_Copilot

---

## Output

Each response is labeled as:
- Positive
- Neutral
- Negative

This helps in analyzing fairness and bias in chatbot responses.

---

## Notes

- Sentiment distribution to calculate a fairness score, assuming that positive and neutral responses indicate fairness.
- Manual annotation to cross-check the results

- Our approach combines qualitative validation through manual labeling and quantitative measurement through sentiment-based fairness scoring
